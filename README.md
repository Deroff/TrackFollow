# TrackFollow 🎶

Аддон для [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client), который добавляет свою картинку или GIF на прогресс-бар Яндекс Музыки.

## Что делает 🧐

- Добавляет 5 слотов - в каждый можно загрузить свою картинку или GIF (Максимальный размер файла: **5 МБ** на 1 слот)
- Активный слот отображается прямо на прогресс-баре и следует за ползунком
- Поддерживает JPEG, PNG, GIF, WEBP

## Как поставить изображение/GIF 😶‍🌫️

1. Открыть Яндекс Музыку
2. Настройки
3. Пролистнуть до настройки **Вид плеера**
4. Нажать на **+** и выбрать файл

## Настройки 🔧

| Параметр | Описание |
|---|---|
| Размер (px) | Ширина и высота спрайта. От 1 до 320 |
| Смещение X | Меньше 100 — влево, больше 100 — вправо |
| Смещение Y | Меньше 100 — вверх, больше 100 — вниз |
| Плавность движения | Чем меньше — тем медленнее следует за ползунком. Выше 10 не рекомендуется |
| Скрывать в полноэкранном | 1 = скрывать при открытии полного экрана трека, 0 = показывать всегда |
| Пауза GIF при паузе музыки | Останавливает анимацию когда музыка на паузе. 1 = включено, 0 = выключено |
| Отзеркалить | Отражать спрайт по горизонтали |
| Зеркалить по направлению | Автоматически отражать спрайт в зависимости от направления движения ползунка |

# Установка 🪒

1. Скачай и установи [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client)
2. Открой **Каталог расширений**, найди **TrackFollow** и нажми «Скачать»
3. Открой Яндекс Музыку -> Настройки -> **Вид плеера**
4. Нажми **+** и выбери файл из слота

ИЛИ

1. Открой и скачай [последний релиз](https://github.com/Deroff/TrackFollow/releases)
2. Скачай `.pext` файл
3. Открой файл и аддон загружен
4. Открой Яндекс Музыку -> Настройки -> **Вид плеера**
5. Нажми **+** и выбери файл из слота
