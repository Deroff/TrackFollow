# TrackFollow

Аддон для [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client), который добавляет свою картинку или GIF на прогресс-бар Яндекс Музыки. Картинка плавно следует за ползунком воспроизведения.

## Что делает

- Добавляет 5 слотов - в каждый можно загрузить свою картинку или GIF
- Активный слот отображается прямо на прогресс-баре и плавно следует за ползунком в реальном времени
- Поддерживает JPEG, PNG, GIF, WEBP - включая анимированные GIF

## Как поставить изображение/GIF
- Открыть Яндекс Музыку
- Настройки
- Пролистнуть до настройки Вид плеера
- нажать на +

## Настройки

| Параметр | Описание |
|---|---|
| Размер (px) | Ширина и высота спрайта. От 1 до 320 |
| Смещение X | Меньше 100 - влево, больше 100 - вправо |
| Смещение Y | Меньше 100 - вверх, больше 100 - вниз |
| Плавность движения | Чем меньше тем медленнее идёт за плеером и наоборот. (Выше 10 лучше не делать) |
| Скрывать в полноэкранном | 1 = скрывать при открытии полного экрана трека, 0 = показывать всегда |
