# TrackFollow 🎶

Аддон для [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client), который добавляет свою картинку или GIF на прогресс-бар Яндекс Музыки.

## Что делает 🧐

- До **12 слотов** - в каждый можно загрузить свою картинку или GIF (макс. **5 МБ** на слот)
- Активный слот отображается прямо на прогресс-баре и плавно следует за ползунком
- Плавная анимация при переключении между слотами
- Поддерживает JPEG, PNG, GIF, WEBP

## Как добавить изображение/GIF 😶‍🌫️

1. Открыть Яндекс Музыку
2. Настройки
3. Пролистать до настройки **Вид плеера**
4. Нажать **+** на нужном слоте и выбрать файл

## Настройки 🔧

| Параметр | Описание |
|---|---|
| Размер (px) | Ширина и высота спрайта. От 1 до 320 |
| Смещение X | Меньше 100 - влево, больше 100 - вправо |
| Смещение Y | Меньше 100 - вверх, больше 100 - вниз |
| Плавность движения | Чем меньше значение - тем медленнее следует за ползунком. Выше 10 не рекомендуется |
| Скрывать в полноэкранном | Скрывать спрайт при открытии полного экрана трека |
| Пауза GIF при паузе музыки | Останавливает анимацию когда музыка на паузе. замораживает паузный спрайт или основной если нету паузного |
| Отзеркалить | Отражать спрайт по горизонтали |
| Зеркалить по направлению | Автоматически отражает спрайт, когда ползунок движется влево |
| Рандомный слот при смене трека | При смене трека автоматически выбирается случайный слот из заполненных |


## Установка 🪛

### Через каталог PulseSync (рекомендуется)

1. Скачай и установи [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client)
2. Открой **Каталог расширений**, найди **TrackFollow** и нажми «Скачать»
3. Открой Яндекс Музыку → Настройки → **Вид плеера**
4. Нажми **+** на нужном слоте и выбери файл

### Вручную через .pext

1. Скачай [последний релиз](https://github.com/Deroff/TrackFollow/releases)
2. Скачай `.pext` файл (Также требуется [PulseSync](https://github.com/PulseSync-LLC/PulseSync-client))
3. Открой файл - аддон установится автоматически
4. Открой Яндекс Музыку → Настройки → **Вид плеера**
5. Нажми **+** на нужном слоте и выбери файл
